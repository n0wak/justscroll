justScroll is a simple dependency-free javascript library for scrolling the page to somewhere.

You can scroll to an id using justScroll.to('elementID');

You can scroll to any point on the page using justScroll.to(1500);

You can can control the speed by passing it the time in seconds justScroll.to('sec5', 2);

There are some build in easing functions too justScroll.to('sec6', 1, 'easeOutQuad');

justScroll is on github and created by n0wak . Background pattern from The Pattern Library and Easing Functions sourced from greweb. Tested in Chrome 38, Safari 6.1.3, and IE 10+.
